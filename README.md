##  技术路线：
- 以89C51为系统的MCU，通过GPS模块获取星历数据，通过串口方式发送至单片机处理；
- MCU负责响应用户的指令、解析星历数据、驱动电机等任务；
- 手机端APP通过Bluetooth蓝牙方式发送特定的指令实现开锁、获取定位信息、报警等业务处理；
- 机械锁部分通过步进电机驱动开锁装置，执行开锁动作。
